📘 User Guide

Basic Interaction

  • Input: Click buttons or use your keyboard.
  • Operations: Supports +, −, ×, and ÷.

Advanced Display & Logic

  • Dynamic View: Equation builds live. Hit = (or Enter) to move history to the top.
  • Active Operator: The selected math symbol stays highlighted.

Corrections & Errors

  • Delete (⌫): Use the backspace to remove the last character or operator.
  • All Clear (AC): Use AC (or Escape) to reset.
  • Error Handling: Detects invalid math (e.g., dividing by zero).
